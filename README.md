# shooting-game

一款使用 Ebiten 开发的简易射击游戏

![game](./images/game.png)

- ebiten：ebiten 引擎的一些基础用法
- images：游戏中用到的一些图片
- bullet：子弹
- config：游戏配置
- game：游戏主逻辑
- monster：怪物👹
- main：启动
